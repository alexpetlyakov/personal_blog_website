# Personal Blog Website

## About the project
This is a simple blog website made as a simple exercise in html and css.

## License
Not for commercial use.

## Contents
This project consists of the following:
  * index.html - home page
  * post.html - example blog post page
  * post_missing.html - placeholder for missing pages errors (mostly to handle placehodlers for blog posts)
 * /fonts - custom font files
 * /css - stylesheets
 * /img - image assets (logos, images used across pages etc/)
