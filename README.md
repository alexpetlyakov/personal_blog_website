# Personal Blog Website

## About the project & technologies
This is a simple blog website made as a simple exercise in html and css. It uses HTML5 and only standard CSS, no JS components.

## License
Not for commercial use.

## Instructions
Clone the repo and launch index.html

## Contents
This project consists of the following:
  * index.html - home page
  * /html - hosts html other than homepage
     * post.html - example blog post page
     * post_missing.html - placeholder for missing pages errors (mostly to handle placehodlers for blog posts)
  * /fonts - custom font files
  * /css - stylesheets
    * style.css - main stylesheet. covers most of the styling and imports supporting css files
    * footer.css
    * header.css
    * post.css - styling for blog post page
    * post_missing.css - styling that covers missing pages
  * /img - image assets (logos, images used across pages etc/)
